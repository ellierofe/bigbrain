## Decisions made
- **AI SDK v6 API changes:** `maxSteps` replaced by `stopWhen: stepCountIs(5)`. `toDataStreamResponse()` doesn't exist; using `toUIMessageStreamResponse()` for tool call streaming. Zod 4 schemas need `zodSchema()` wrapper for type inference with `tool()`.
- **FalkorDB vector search deferred:** v1 uses pgvector only. FalkorDB handles traversal. Backlogged as VEC-03 — evaluate after RET-01 v1 at scale.
- **Analysis commit mapping:** Themes → Concept nodes, insights/realisations → Idea nodes, methodology → Methodology nodes. Operational items (recurring blockers, commitments) not committed to graph — they're session management, not knowledge.
- **Graph junk cleared:** 60 nodes from early INP_03 runs were low-quality headline-style extractions. Cleared rather than trying to salvage. Starting fresh with the improved extraction prompts.
- **Embeddings at write time:** `writeNode()` now handles embedding generation. The manual embedding generation in `commitExtraction()` is technically redundant but harmless — cleanup deferred.
