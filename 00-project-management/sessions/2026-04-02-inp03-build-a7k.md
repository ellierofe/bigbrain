## Decisions made

- `src_source_documents.type` is content/intent type, not file format. Format stays in `mimeType`.
- `fileUrl` nullable — paste inputs are valid sources without a file.
- Tags are free-form, stored on `src_source_documents.tags`, surfaced in the process UI metadata row.
- Date node temporal edges always written — no opt-out. Commit date used as fallback.
- Stories commit to Postgres only (no graph node) in v1 per brief. Subject defaults to 'self', editable per item.
- base-ui Checkbox doesn't support `indeterminate` — section-level checkbox uses boolean (checked = all selected, unchecked = not all selected). Clean visual, no indeterminate state.
- `BRAND_ID` hardcoded per existing convention — replace with session lookup when multi-brand auth lands.

## What came up that wasn't planned

- `zod` was already present as a transitive dep from `@ai-sdk` — `npm install` just hoisted it.
- base-ui `Checkbox` (not radix) doesn't support `'indeterminate'` — discovered at tsc check, fixed immediately.
- Date node lookup requires querying Neon mirror by `name` + label + `properties->>'level'` since seed used randomUUID() for node ids.
